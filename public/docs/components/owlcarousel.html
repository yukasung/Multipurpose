    Owl Carousel

    Owl Carousel is a touch enabled jQuery plugin that lets you create beautiful responsive carousel sliders.

    Basic

    The following example shows how to create a basic carousel.

    With controls

    Adding in the previous and next controls.

    Use the .owl-nav-dark class to make controls dark color.

    Hover controls

    Show the previous and next on hover over by adding .owl-nav-md-hover

    Dots navigations

    Change dots navigation color by adding .owl-dots-primary or .owl-dots-white.

    Dots inside

    Add .owl-dots-inside for show dots navigation inside the owl carousel.

    For a complete reference of all owl carousel options, methods and events, go to our Owl Carousel Reference
