    Overflow

    The overflow classes specifies when content is clipped if necessary to fit the padding box.

    Classes

    Class Description
    .overflow-hidden The overflow is clipped, and the rest of the content will be invisible.
    .overflow-hidden-y The content is clipped when it overflows at the top and bottom edges.
    .overflow-hidden-x The content is clipped when it overflows at the left and right edges.
